# AGENTS.md

## 역할
- `docs/`는 설계와 운영 기준을 담는 문서 폴더다.
- 코드 변경의 의미를 설명하는 source of truth로 취급한다.

## 작업 규칙
- 데이터 모델, 용어, selection/view 규칙이 바뀌면 관련 문서를 같은 변경에서 갱신한다.
- 현재 문서 일부는 계획 단계 구조를 반영하고 있어 실제 코드와 차이가 날 수 있다. 차이를 발견하면 숨기지 말고 문서에 반영한다.
- 구현 세부를 그대로 복붙하기보다 "왜 이 경계가 필요한지"를 설명하는 쪽을 우선한다.

## 확인
- 문서만 바꿀 때도 `README.md` 또는 관련 `AGENTS.md`와 모순이 없는지 확인한다.
